# Youtube-Channel-Data-Scrapper-python-
Python code for channel data scrapping for any purpose. The output file will contain Video title, views, published date and URL.

Just copy the file fetch_youtube_videos1.py

Run this first:
```bash
pip install requests
```
then run:
```bash
python fetch_youtube_videos1.py
```
# Where to Find API key🤔
-> Go to console.cloud.google.com    
-> Create a project → Enable YouTube Data API v3   
-> Go to Credentials → Create API Key

# To run
Since you got your API Key, now paste into the code and also paste the channel ID you want to scrap the data of and run the commands above given.
That's it guys.
